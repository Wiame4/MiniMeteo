Informations API

Endpoints disponibles:

  • /api/weather - Météo actuelle
  • /api/forecast - Prévisions sur 24h

Paramètres:

  • city: Nom de la ville (ex: Paris)
  • units: Unités (metric pour °C, imperial pour °F)
  • lang: Langue (fr, en, es, etc.)

Exemple: /api/weather?city=Paris&units=metric&lang=fr
